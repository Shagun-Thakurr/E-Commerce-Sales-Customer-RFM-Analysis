## Project Overview
This project analyzes e-commerce sales performance and customer behavior using the Superstore dataset.
I performed data cleaning and feature engineering in Python using jupyter notebook, executed business-focused SQL queries, applied RFM segmentation for customer intelligence, and built an interactive dashboard for insights.


## Objectives
* Analyze sales, profit, and regional performance.
* Identify top products and categories.
* Segment customers using RFM (Recency, Frequency, Monetary).
* Provide data-driven business recommendations.

  
## Tools Used
* **Python:** Pandas, NumPy, Matplotlib, Seaborn.
* **SQL:** Aggregations, KPIs, and business-focused queries.
* **Power BI:** Interactive dashboard for visual insights.

  
## Key Work Performed
* Analyzed a comprehensive dataset of 9,994 transactions across multiple regions and categories.
* Performed sales and profitability analysis.
* Created RFM scores and customer segments (VIP customer,Loyal customer, At Risk, etc.).
* Built interactive dashboard with KPIs, trends, and segmentation insights.
* performed year-wise sales & profit analysis.

  
## Outcome
* This project demonstrates end-to-end data analysis skills — from raw data to business insights — combining Python, SQL, and BI tools to support strategic decision-making.
